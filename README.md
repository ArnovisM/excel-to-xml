# Excel to XML Converter

This tool allows you to easily convert your Excel shipping manifests into XML format. It supports converting multiple files at once and saving them to a custom location.

## Instructions

1.  **Start the Application**:
    *   Double-click the **`run.bat`** file.
    *   This will install necessary dependencies (first time only) and open the converter window.

2.  **Select Files**:
    *   Click the **"Select Files"** button.
    *   Select one or more Excel files (.xlsx) you want to convert.

3.  **Select Output Folder (Optional)**:
    *   By default, XML files are saved in the same folder as the input files.
    *   To save them elsewhere, click **"Select Folder"** and choose a destination.

4.  **Convert**:
    *   Click the **"Convert All to XML"** button.
    *   A success message will appear when finished, showing how many files were converted.

## Requirements

*   Python installed on your computer.
*   Internet connection (only for the first run to install libraries).

## Support

If you encounter any issues, please ensure your Excel file is not corrupted and follows the expected format.
