# Excel to XML Converter

This tool allows you to easily convert your Excel shipping manifests into XML format.

## Instructions

1.  **Start the Application**:
    *   Double-click the **`run.bat`** file.
    *   This will install necessary dependencies (first time only) and open the converter window.

2.  **Convert File**:
    *   In the converter window, click the **"Select Excel File"** button.
    *   Browse and select your Excel file (.xlsx).
    *   Click the **"Convert to XML"** button.

3.  **Result**:
    *   Once the conversion is finished, a success message will appear.
    *   The XML file will be saved in the same folder as your original Excel file, with the same name but `.xml` extension.

## Requirements

*   Python installed on your computer.
*   Internet connection (only for the first run to install libraries).

## Support

If you encounter any issues, please ensure your Excel file is not corrupted and follows the expected format.
